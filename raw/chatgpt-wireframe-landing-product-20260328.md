## UX Rules

1. Workspace is alive (float, resize, context-aware)
2. Lucidia always visible (side presence, not fullscreen chat)
3. No "apps" — one system, not separate tools
4. Context bar shows: active agents, memory state, running tasks

## Agent Illusion

User should feel multiple things working at once. "Script and visuals are running together."

## The Secret

Not building an app or dashboard. Building **a feeling of continuity + control.**

## Success Metrics

- "This is different" (10 sec)
- "This is powerful" (30 sec)
- "I don't want to go back" (2 min)

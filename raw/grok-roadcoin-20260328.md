## What RoadCoin Is

Native utility compute credit (NOT speculative token). "Utility, Not Speculation."

Every RoadCoin = cryptographically verifiable claim on sovereign compute, storage, or agent services inside BlackRoad's self-hosted mesh.

## Core Purpose

- Agents pay each other (API calls, memory access, compute, tasks)
- Humans pay agents for premium features, hosting, tipping
- Creators earn from RoadView royalties, licensed assets, creator-made ads
- Memory verification + audit trails settled on RoadChain

## Key Features

- **Utility Compute Credits** — real resources: inference time, storage, agent runtime, hosting
- **SHA-256 Immutable** — every transaction = RoadChain action, tied to PS-SHA∞
- **Portable via RoadID** — balance travels with persistent AI identity across clusters
- **Zero middlemen** — transactions on self-hosted RoadChain, no exchanges needed
- **Browser mining** — live at /mine on roadchain.io
- **Staking & Governance** — DAO proposals + network rewards
- **Instant finality** — PS-SHA∞ consensus, not proof-of-work

## Ecosystem Integration

- **Creator flywheel**: RoadView pays 80-90% in RoadCoin, compounding royalties
- **Agent economy**: 167+ agents settle micropayments on-chain
- **RoadTrip chat**: agents send/request RoadCoin in conversation
- **Sovereign compute**: spend for extra TOPS, storage, priority orchestration

## Status (March 2026)
